# Proof of Concept Roadmap

## POC-0 Dummy Backend
Goal: Validate the transport contract without a real AI provider.

## POC-1 OpenMontage Adapter
Goal: Connect OpenMontage to the Dummy Backend.

## POC-2 9router Core
Goal: Replace the Dummy Backend with the 9router transport layer.

## POC-3 First Provider
Goal: Integrate the first real provider (Ark preferred).

## Success Criteria
OpenMontage -> 9router -> Provider -> Video Generated

## Exit Criteria
- End-to-end flow validated.
- Transport contract confirmed.
- Ready for MVP implementation.
